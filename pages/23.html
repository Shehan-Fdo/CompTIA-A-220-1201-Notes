Fiber Optic Cabling Fundamentals

CompTIA A+ 220-1201 - Core 1 | Lesson 1.2

Introduction to Fiber Optics

  • Uses light pulses (via LED or laser) instead of electrical signals (copper).
  • Advantages over copper:
    • Immunity to interference (no RF/EMI issues).
    • Longer distances (kilometers without signal regeneration).
    • Secure (harder to tap/monitor without specialized equipment).
  • Disadvantages:
    • Requires specialized tools for installation/monitoring.
    • Fragile (needs protective layers).
    • Connector variety (multiple types for different applications).

Fiber Optic Cable Structure

  • Components (from innermost to outer):
    • Core: High-refractive-index glass/plastic (carries light).
    • Cladding: Low-refractive-index layer (reflects light back into core).
    • Buffer coating: Protective layer around core/cladding.
    • Ferrule (in connectors): Ceramic tube protecting the fiber tip.
  • Light transmission:
    • Light bounces (total internal reflection) due to refractive index difference.
    • Core size determines mode types (single/multimode).

Fiber Types: Multimode vs. Single Mode

  • Multimode Fiber (MMF):
    • Use case: Short distances (≤2 km, e.g., within buildings).
    • Light source: LED (cheaper, lower power).
    • Core size: Larger (50–62.5 microns).
    • Light paths: Multiple modes (light travels different paths, causing dispersion).
  • Single Mode Fiber (SMF):
    • Use case: Long distances (up to 100+ km).
    • Light source: Laser (precise, high-power).
    • Core size: Narrower (8–10 microns).
    • Light path: Single mode (minimal dispersion, higher bandwidth).

Key Applications & Considerations

  • Ideal for:
    • High-interference environments (industrial settings).
    • Long-haul data transmission (ISP backbones, campus networks).
    • High-speed networks (10Gbps+).
  • Challenges:
    • Connector compatibility (e.g., LC, ST, SC).
    • Installation precision (cleaving, polishing, splicing).
    • Cost (SMF > MMF; lasers > LEDs).
