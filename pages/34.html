Motherboard Form Factors

CompTIA A+ 220-1201 - Core 1 | Lesson 3.5

Introduction to Motherboard Types

  • Computing platforms vary in size (e.g., desktop, small office, media center) but share core components: CPU, memory, storage, network connections.
  • Key differentiators between motherboards:
    • Physical size (footprint).
    • Component layout (larger boards allow more components).
    • Power connections (standardized but may vary, e.g., 20-pin vs. 24-pin ATX).
    • Airflow/cooling (heat management varies by size).
  • For A+ Certification (220-1201), focus on 3 form factors:
    • ATX (standard).
    • micro-ATX (compact).
    • mini-ITX (ultra-small).

ATX (Advanced Technology Extended)

  • Standard since 1995, with updates over time.
  • Features:
    • Largest of the three form factors.
    • Expansion slots: Typically 4+ (e.g., PCIe, RAM slots).
    • Power connector: 24-pin (modern) or 20-pin (older).
    • Mounting points: Standardized for case compatibility.
    • Use case: High-performance systems (e.g., video editing, gaming).
  • Example: Gigabyte GA-P67A-UD3P-B3 (4 RAM slots, multiple expansion slots).

micro-ATX

  • Smaller than ATX but retains key compatibilities:
    • Same mounting points as ATX.
    • Same power connectors (24-pin).
  • Trade-offs:
    • Fewer expansion slots (e.g., 1–2 PCIe).
    • Limited RAM slots (typically 2).
  • Use case: Balanced systems (e.g., small office, home theater PCs).
  • Example: MSI H81M-P33 (1 expansion slot, 2 RAM slots).

mini-ITX

  • Smallest form factor (ultra-compact).
  • Features:
    • Same mounting points as ATX (compatible with larger cases).
    • Minimal expansion: Often 1 slot (if any).
    • Power: Standard 24-pin ATX connector.
  • Use case: Space-constrained environments (e.g., media centers, single-task devices).
  • Example: mini-ITX boards (e.g., for streaming devices near TVs).

Comparison Summary

  • Size: ATX (largest) → micro-ATX → mini-ITX (smallest).
  • Expansion:
    • ATX: 4+ slots.
    • micro-ATX: 1–2 slots.
    • mini-ITX: 0–1 slot.
  • RAM Slots:
    • ATX: 4.
    • micro-ATX: 2.
    • mini-ITX: 1–2.
  • Compatibility: All share ATX mounting points and power standards.
  • Scenario-Based Selection:
    • High-performance: ATX.
    • Balanced/compact: micro-ATX.
    • Ultra-small/single-task: mini-ITX.

A+ Exam Focus (3.5)

  • Objective: "Install and configure motherboards, CPUs, and add-on cards" given a scenario.
  • Key Skill: Select the appropriate form factor for a use case (e.g., media center vs. video editing).
  • Memorize:
    • Three form factors: ATX, micro-ATX, mini-ITX.
    • Differences in size, expansion, and use cases.
