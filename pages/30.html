Understanding RAM (Random Access Memory)

CompTIA A+ 220-1201 - Core 1 | Lesson 1.2

RAM Overview

  • RAM (Random Access Memory) is temporary, high-speed storage used for executing applications and calculations.
  • Data must be loaded into RAM for the computer to process it.
  • RAM is volatile (cleared when power is off) and not long-term storage (unlike HDDs or SSDs).
  • Common workflow: Load data from storage → Process in RAM → Save results back to storage.

Memory Module Types

  • DIMM (Dual Inline Memory Module):
    • Used in desktops.
    • Has two sets of electrical contacts (one per side).
    • Installed vertically in motherboard slots (push to insert, release side locks to remove).
  • SO-DIMM (Small Outline DIMM):
    • Used in laptops/mobile devices (~half the size of DIMM).
    • Installed horizontally (slide in, push down to lock).

Memory Technology

  • DRAM (Dynamic RAM):
    • Stores data in capacitors (requires constant refreshing).
    • Allows instantaneous access to any address (no sequential seeking like tapes).
  • SDRAM (Synchronous DRAM):
    • Synchronized with the system clock for standardized data transfer rates.
    • Enables coordination between CPU and memory.
  • Data Width: Memory transfers data in 64-bit blocks.

DDR Memory Evolution

  • DDR (Double Data Rate):
    • Transfers data on both rising and falling edges of the clock cycle (doubles transfer speed vs. single data rate).
    • Versions are not backward-compatible:
      • DDR2 → DDR3 → DDR4 → DDR5 (each improves speed/capacity).
      • Physical key notches prevent incorrect installation (e.g., DDR4 won’t fit in a DDR3 slot).

Key Considerations

  • Performance Impact:
    • Faster memory → Faster system performance.
    • Memory speed must be compatible with the motherboard.
  • Upgrade Tips:
    • Check motherboard documentation for supported memory types (e.g., DDR4, max capacity).
    • Mixing different DDR versions or speeds may cause instability.
