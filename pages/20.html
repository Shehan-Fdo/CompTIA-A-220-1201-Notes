Display Specifications

CompTIA A+ 220-1201 - Core 1 | Lesson 1.2

Display Use Cases

  • Determine display purpose before selecting specifications:
    • Gaming - High refresh rates, low input lag
    • Video presentations - Color accuracy, resolution
    • Lobby displays - Viewing angles, brightness
    • Office work - Text clarity, ergonomics

Pixel Density (PPI)

  • Measures pixels per inch (or cm) - determines display sharpness
  • Higher PPI = crisper image (more pixels in same physical space)
  • Calculation: Horizontal Pixels ÷ Display Width (inches)
  • Examples:
    • 27" 4K (3840×2160) on 24" width = 160 PPI
    • 65" 4K (3840×2160) on 57" width = 67 PPI
  • Consider DPI (dots per inch) for printed output compatibility

Refresh Rate (Hz)

  • Measures screen updates per second (Hz = cycles/second)
  • Often equals FPS (frames per second) when V-Sync is enabled
  • Common requirements:
    • Movies: 24Hz
    • TV/online video: 30Hz
    • Gaming/sports: 60Hz+ (120Hz/144Hz for competitive)
  • Low refresh rates cause stuttering in fast-moving content
  • Requires compatible:
    • GPU (video card)
    • Connection type (HDMI 2.1/DisplayPort 2.1 for 4K@144Hz)

Resolution Standards

  • Defines horizontal × vertical pixels:
    • HD: 1920×1080
    • 4K/UHD: 3840×2160 (4× HD pixels)
    • Non-standard resolutions exist (check specs)
  • Higher resolution = sharper image on same-sized displays
  • Common aspect ratio: 16:9 (widescreen)

Color Gamut

  • Range of colors a display can reproduce vs. human eye
  • Measured against standards:
    • sRGB (95%-100% ideal for general use)
    • Adobe RGB (wider gamut for photography)
    • DCI-P3 (cinema standard, ~98% coverage)
    • Rec. 709 (HDTV standard)
  • Visualized on CIE 1931 color space chart
  • Technology impact:
    • OLED: Best color accuracy/black levels
    • IPS LCD: Good color but lower contrast than OLED
    • TN LCD: Poorest color accuracy
  • Critical for:
    • Graphic design
    • Video editing
    • Photography

Connection Standards

  • Bandwidth affects supported resolution/refresh rates:
    • HDMI 2.1: 4K@144Hz max
    • DisplayPort 2.1: Dual 4K@144Hz
    • Older standards (HDMI 2.0/DisplayPort 1.4) limit high-refresh 4K
  • Match display capabilities with GPU and cable specs

End of Notes
