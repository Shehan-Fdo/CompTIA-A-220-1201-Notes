Mobile Device Networks & Connectivity

CompTIA A+ 220-1201 - Core 1 | Lesson 1.2

Cellular Networks Overview

  • Cellular networks divide land into cells using antennas for coverage.
  • Supports both voice and data communication.
  • User controls include:
    • Toggle cellular data (on/off).
    • Airplane mode disables all wireless communication.

Evolution of Cellular Technology

  • 3G (1998): Enabled GPS, mobile TV, video conferencing.
  • 4G (LTE):
    • Based on GSM (Global System for Mobile Communications).
    • Bandwidth: Up to 150 Mbps.
    • LTE Advanced (LTEA): Up to 300 Mbps.
  • 5G (2020):
    • High-frequency: Up to 10 Gbps.
    • Low-frequency: 100–900 Mbps.
    • Enables IoT with high-speed data transfer.

Wi-Fi (802.11) Connectivity

  • High-speed wireless for home/office.
  • Supports voice, video, data.
  • Limited range; requires local access point.
  • Features:
    • Wi-Fi calling: Use Wi-Fi for voice if cellular signal is weak.
    • Hotspot: Phone acts as a router (requires carrier support).

SIM Cards

  • Subscriber Identity Module (SIM):
    • Identifies device on the network.
    • Stores carrier info, phone number, contacts.
    • Physical SIM can be transferred between devices.
  • eSIM (Embedded SIM):
    • Software-managed; no physical card.
    • Supports QR code or digital transfer.
  • Multi-SIM support: Some phones allow multiple profiles.

Bluetooth Pairing

  • Process:
    • Enable Bluetooth on both devices.
    • Set devices to discoverable mode.
    • Confirm PIN (static or dynamic).
  • Post-pairing: Auto-reconnects after initial setup.
  • Check device documentation for PIN nuances.

GPS & Location Services

  • Global Positioning System (GPS):
    • US DoD technology using satellites.
    • Requires 4+ satellites for accuracy.
    • Provides longitude, latitude, altitude.
  • Alternative location methods:
    • Wi-Fi networks.
    • Cellular triangulation.
