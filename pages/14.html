IPv4 and IPv6 Fundamentals

CompTIA A+ 220-1201 - Core 1 | Lesson 1.4

Introduction to IP Addressing

  • IPv4 is the most widely used protocol for network communication.
  • A+ exam objectives require knowledge of configuring systems with IPv4 and IPv6 parameters.
  • Both versions share operational similarities but differ in address format and configuration methods.

IPv4 Address Structure

  • 32-bit address represented as four decimal octets (e.g., 192.168.1.131).
  • Each octet is 8 bits (1 byte) with a value range of 0–255.
  • Total combinations: 4.29 billion addresses.
  • Binary representation: Each octet can be converted to 8 binary digits (e.g., 11000000.10101000.00000001.10000011).

IPv4 Address Limitations and Solutions

  • Limited addresses (4.29B) vs. 20B+ connected devices.
  • Network Address Translation (NAT) enables private networks to share a single public IP.
  • Private IP ranges (defined in RFC 1918):
    • 10.0.0.0–10.255.255.255 (16M+ addresses, corporate use).
    • 172.16.0.0–172.31.255.255 (1M+ addresses, medium networks).
    • 192.168.0.0–192.168.255.255 (65K+ addresses, home networks).

IPv6 Address Structure

  • 128-bit address (vs. IPv4’s 32-bit), enabling 340 undecillion unique addresses.
  • Represented in hexadecimal with colons (e.g., FE80:0000:0000:0000:5DEL:0652:CHFX:8F52).
  • Simplified notation:
    • Leading zeros in a 16-bit block can be omitted (e.g., :0652: → :652:).
    • Consecutive blocks of zeros replaced with :: (once per address).
  • Default subnet mask: /64 (first 64 bits = network prefix, last 64 bits = host ID).

Key Differences: IPv4 vs. IPv6

  • Address Length: IPv4 = 32-bit, IPv6 = 128-bit.
  • Representation: IPv4 = decimal, IPv6 = hexadecimal.
  • Address Space: IPv6 eliminates scarcity with 340 undecillion addresses.
  • Subnetting: IPv6 rarely requires subnetting due to vast address availability.
  • DNS Reliance: IPv6 addresses are complex; DNS is critical for usability.
  • NAT: Less critical in IPv6 due to abundant addresses.

Practical Implications

  • IPv4 remains dominant but is supplemented by NAT and private addressing.
  • IPv6 adoption grows to support IoT and future-proof networking.
  • Exam focus: Configure both versions, understand private/public addresses, and subnetting basics.
