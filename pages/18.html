Cable and Wireless Network Tools

CompTIA A+ 220-1201 - Core 1 | Lesson 2.2

Cable Connectors and Crimping

  • Common cable types requiring connectors: coax, twisted pair, and fiber.
  • Crimping is used to attach connectors permanently to cables using a cable crimper.
  • For twisted pair (e.g., RJ45):
    • Connector pins pierce wire insulation to contact copper.
    • Crimping secures the cable and ensures cable stay prevents pull-out.
    • Tools needed: crimper, electrician scissors, wire stripper.
  • Custom cables can be created with practice, allowing flexible lengths.

Wireless Network Analysis

  • Wireless advantages: No physical wires, but challenges include interference and signal strength.
  • Tools for analysis:
    • Wi-Fi analyzer: Shows channels, signal strength, interference, and connected devices.
    • Spectrum analyzer: Detects all frequency usage (including non-802.11 devices).
    • Signal strength impacts throughput; weaker signals near noise floor reduce performance.

Cable Tracking and Testing

  • Tone generator and inductive probe:
    • Generates an analog signal on a cable.
    • Probe detects signal without physical contact (inductive coupling).
    • Useful for identifying cables in large bundles (e.g., data centers).
  • Punchdown blocks:
    • Terminates multiple cables for cross-connections.
    • Maintain wire twists to prevent interference.
    • Numbered for documentation and tracing.
  • Cable testers:
    • Verifies pin-to-pin continuity (e.g., RJ45 1→1, 2→2).
    • Detects miswired or broken connections.
    • Basic continuity test (not signal quality).
  • Loopback plugs:
    • Tests physical interfaces (serial, RJ45, fiber).
    • Loops sent data back to receive interface.
    • Diagnoses hardware faults (not cable issues).

Network Traffic Capture

  • Methods to gather packets:
    • Physical tap:
      • Breaks a link to copy traffic.
      • Passive (no power) for fiber; active for copper.
      • Permanent installation for frequent monitoring.
    • Port mirroring (SPAN):
      • Switch feature to copy traffic from one port to another.
      • Non-disruptive; no physical tap needed.
      • Also called port redirection or switched port analyzer.
