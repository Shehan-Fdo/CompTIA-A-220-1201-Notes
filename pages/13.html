Network Devices

CompTIA A+ 220-1201 - Core 1 | Lesson 1.2

Routers

  • Devices that forward packets between different IP subnets based on destination IP address.
  • Contains a routing table to determine the best path for traffic.
  • Operates at OSI Layer 3 (Network Layer).
  • Can connect different network types (e.g., Ethernet to Wireless, Fiber to Copper).
  • Layer 3 Switches: Combine routing and switching in one device.

Switches

  • Connects end devices and forwards traffic based on destination MAC address.
  • Operates at OSI Layer 2 (Data Link Layer).
  • Uses ASICs (Application-Specific Integrated Circuits) for high-speed switching.
  • Types:
    • Unmanaged Switches: Plug-and-play, no configuration, single VLAN, no SNMP/logs, lower cost.
    • Managed Switches: Supports VLANs, QoS, redundancy, port mirroring, SNMP, and remote management.
    • Layer 3 Switches: Combines switching and routing functionalities.
  • Port counts vary: 24/48 ports (workgroup), hundreds of ports (core).
  • Supports Power over Ethernet (PoE) for devices like phones, APs, and cameras.

Access Points (APs)

  • Connects wireless devices to a wired network.
  • Operates as a bridge (no IP translation or routing).
  • Forwards traffic based on destination MAC address (like a switch).
  • Used in enterprise networks for wireless coverage (vs. home wireless routers).

Patch Panels & Cabling

  • Patch Panels: Centralizes and organizes Ethernet cabling in a network closet.
  • Features:
    • Permanent connections from desks to the panel (punched down).
    • RJ45 connectors on the front for flexible switching.
    • Simplifies moves/adds/changes without rewiring.
    • Labeled ports for easy identification (e.g., "Desk 9").
  • Benefits:
    • Static long-term cabling reduces failures.
    • Short patch cables connect panels to switches for easy reconfiguration.

Firewalls

  • Security devices that filter traffic based on rules.
  • Types:
    • Traditional Firewalls: Filter by port numbers (TCP/UDP).
    • Next-Generation Firewalls (NGFW): Filter by application (e.g., allow web traffic but block remote access tools).
  • Additional Functions:
    • VPN Concentrator: Supports site-to-site or remote access VPNs.
    • Proxy: Inspects traffic between client and server.
    • Routing: Can operate as a router between subnets.
  • Enterprise firewalls may have multiple interfaces for connecting many networks.

Power over Ethernet (PoE)

  • Delivers data and power over a single Ethernet cable.
  • Standards:
    • PoE (802.3af): 15.4W, 350mA (phones, small devices).
    • PoE+ (802.3at): 25.5W, 600mA (PTZ cameras, larger devices).
    • PoE++ (802.3bt): 51W/71.3W, 600mA/960mA (laptops, high-power devices).
  • Types:
    • In-Span: Power provided by the PoE switch.
    • Mid-Span: Power injected via a PoE injector (external device).
  • Use Cases:
    • Wireless access points (ceiling-mounted).
    • VoIP phones.
    • IP cameras (PTZ).
    • Laptops (PoE++).
  • Backward compatible but not upward compatible (e.g., PoE++ device won’t work with PoE switch).

Internet Connection Devices

  • Cable Modem:
    • Uses coaxial cable (DOCSIS standard).
    • Provides high-speed data (1Gbps+) over TV cable infrastructure.
    • Common in homes and businesses with cable TV.
  • DSL Modem:
    • Uses telephone lines (digital signals alongside analog voice).
    • Speed depends on distance from central office (max ~10,000 ft).
    • Typical speeds: 200Mbps downstream, 20Mbps upstream.
  • Optical Network Terminal (ONT):
    • Converts fiber optic to copper (Ethernet/RJ45).
    • Acts as the demarcation point (divides ISP and user responsibility).
    • Supports voice (VoIP), data (Ethernet), and video (coax).

Network Interface Cards (NICs)

  • Hardware that connects devices (PCs, servers) to a network.
  • Types:
    • Built into motherboards (laptops/desktops).
    • Expansion cards (additional ports, e.g., 10Gbps, fiber).
  • Each NIC has a unique MAC address.
  • Supports various speeds/stands:
    • 100Mbps, 1Gbps (copper).
    • 10Gbps (fiber/copper).
