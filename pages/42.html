Expansion Cards

CompTIA A+ 220-1201 - Core 1 | Lesson 1.2

Introduction to Expansion Cards

  • Modern computers use modular design for customization via expansion cards.
  • Expansion cards add additional functionality not available on the motherboard.
  • Designed for end-user installation without manufacturer assistance.
  • Process:
    • Install hardware → Power on → OS auto-detects and installs device drivers.

Types of Expansion Cards

  • Sound Cards:
    • Provides superior audio output vs. onboard audio.
    • Features: high-quality audio, multi-speaker support, input/output options.
    • Inputs: microphones, musical instruments, line-in.
    • Outputs: left/right channels, headphone jack, digital audio.
  • Graphics Cards (GPU):
    • Discrete GPUs offer dedicated processing for high-end tasks.
    • Use cases: gaming, video editing, graphics processing.
    • Connects via PCI Express with dedicated video outputs (HDMI, DisplayPort).
    • Integrated graphics use CPU resources and motherboard outputs (VGA, DVI, HDMI).
  • Video Capture Cards:
    • Enables video input from external sources (cameras, other PCs).
    • High throughput via PCI Express.
    • Supports formats: HDMI, SDI.
  • Network Interface Cards (NICs):
    • Adds wired Ethernet if motherboard lacks it or for multiple connections.
    • Use cases: servers, security devices.
    • Multiport NICs support 4+ Ethernet ports in one slot.

Installation & Configuration

  • Pre-Installation Checks:
    • Review motherboard documentation for compatible slots/interfaces.
    • Check manufacturer specs for hardware/software requirements.
    • Consult knowledge bases or user reviews for insights.
  • Driver Installation:
    • Some cards require driver pre-installation; others post-installation.
    • OS often auto-installs drivers, but verify via Device Manager.
    • Always use latest drivers from the manufacturer’s website.
    • Uninstall old drivers before updating to avoid conflicts.
  • Verification:
    • Use Windows Device Manager to check driver status.
    • Ensure hardware is recognized and functional.
