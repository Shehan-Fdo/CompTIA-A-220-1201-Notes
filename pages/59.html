Display Troubleshooting

CompTIA A+ 220-1201 - Core 1 | Lesson 1.4

Basic Monitor Checks

  • Verify physical connections first:
    • Ensure video cable (HDMI/DisplayPort/VGA/DVI) is fully seated at both ends.
    • Check power cable is connected and monitor receives power (look for standby lights).
  • Common issue: Unplugged or loose cables account for many "no signal" problems.

Input Source Configuration

  • Monitors with multiple inputs (HDMI, DisplayPort, VGA, DVI) may not auto-switch:
    • Manually select correct input via monitor’s OSD (On-Screen Display) menu.
    • Example: If using HDMI, ensure monitor is set to HDMI input.

Brightness/Contrast Issues

  • Symptoms: Dim or washed-out display.
  • Solutions:
    • Adjust brightness/contrast via monitor’s physical buttons or OSD.
    • Check for auto-dimming settings in OS (e.g., battery-saving modes on laptops).

Hardware vs. Software Problems

  • Test with another monitor or another computer to isolate the issue.
  • If monitor works elsewhere, problem is likely with the original computer’s GPU/drivers.
  • If issue persists, monitor hardware may be faulty (e.g., backlight failure, inverter issues).

Boot Process Diagnostics

  • If BIOS/POST messages appear but screen goes black after OS load:
    • Boot into Windows Safe Mode (F8) or VGA Mode (generic 640x480 resolution).
    • Update or roll back GPU drivers.
    • Check for resolution/compatibility issues with the monitor’s native resolution.

Projector-Specific Issues

  • Symptoms:
    • Overheating: Projectors shut down to protect the metal halide bulb (reaches ~1000°C).
    • Cooling fans run post-shutdown to gradually cool the bulb.
    • Bulb failure: Replace bulbs modularly (follow manufacturer guidelines).
  • Maintenance:
    • Clean dust and replace air filters to improve airflow.
    • Ensure proper ventilation (especially for ceiling-mounted units).

Native Resolution & Scaling

  • Native resolution: Fixed pixel count (e.g., 1920x1080) for optimal clarity.
  • Non-native resolutions cause:
    • Blurry text/graphics (scaling artifacts).
    • Stretched or compressed display.
  • Solutions:
    • Set OS to match monitor’s native resolution (e.g., 1920x1080 for Full HD).
    • Use scaling options (e.g., 125%, 150%) for high-DPI displays to improve readability.

Burn-In/Image Sticking

  • Burn-in: Permanent ghosting from static images (e.g., taskbars, logos).
  • Image sticking (LCDs): Temporary ghosting; often reversible.
  • Prevention/Mitigation:
    • Enable pixel shift (monitor setting).
    • Display a white screen overnight to reduce sticking.
    • Avoid static images for prolonged periods.
  • More common in:
    • Older CRT monitors.
    • OLED/LCD panels with prolonged static content (e.g., digital signage).

Dead Pixels

  • Dead pixel: Always black (no power).
  • Stuck pixel: Always red/green/blue (stuck subpixel).
  • Diagnosis:
    • Clean screen to rule out dirt/debris.
    • Use pixel-testing tools (e.g., solid color screens).
  • Solutions:
    • No software fix; requires monitor replacement if under warranty.
    • Minor cases (e.g., edge pixels) may be tolerable.

Flashing/Flickering Screens

  • Causes:
    • Loose/faulty cables (HDMI, DisplayPort, VGA).
    • GPU driver issues (disable hardware acceleration as a test).
    • Monitor hardware failure (e.g., failing backlight, LCD panel).
    • Incompatible resolution/refresh rate.
  • Troubleshooting:
    • Replace cables and test with another monitor.
    • Update or reinstall GPU drivers.
    • Check for physical damage (e.g., cracked LCD).

Color Issues

  • Symptoms: Tinted (too blue/green) or washed-out colors.
  • Solutions:
    • Reset monitor to factory color settings.
    • Adjust tint, RGB balance, or color temperature in monitor OSD.
    • Disable OS night mode/blue light filters (e.g., Windows Night Light).
    • Check GPU driver settings for color profiles.

Audio via Monitor

  • Monitors with built-in speakers may require:
    • Correct audio input source (e.g., HDMI, 3.5mm analog).
    • OS audio output set to monitor (not default speakers).
    • Volume unmuted and adjusted via monitor’s physical controls or OSD.
  • Troubleshooting:
    • Test with another audio source.
    • Ensure cables support audio (e.g., HDMI carries both video/audio; VGA does not).

Backlight & Dim Display

  • Symptoms:
    • Uneven brightness (e.g., one side darker).
    • Flickering backlight.
    • Display visible only at extreme angles (indicates backlight failure).
  • Causes:
    • Failing CCFL (older monitors) or LED backlight.
    • Faulty inverter (for CCFL backlights).
  • Solutions:
    • Replace backlight/inverter (if repairable).
    • Replace monitor if cost-effective.

Display Sizing & Scaling

  • Issues:
    • Underscan: Display too small with black borders.
    • Overscan: Edges of display cut off.
    • Tiny UI elements on high-res displays (e.g., 4K).
  • Fixes:
    • Adjust scaling in OS (e.g., Windows: 125%, 150%, 200%).
    • Enable GPU scaling (e.g., AMD/NVIDIA control panel).
    • Set native resolution and use OS-level zoom for readability.

Advanced Troubleshooting

  • Test Patterns:
    • Use color/grid test patterns to identify:
      • Dead/stuck pixels.
      • Backlight bleeding.
      • Color accuracy issues.
  • Driver Settings:
    • Disable hardware acceleration if causing artifacts.
    • Update or roll back GPU drivers.
  • Hardware Tests:
    • Swap GPU (if discrete) or test with integrated graphics.
    • Check for physical damage (e.g., cracked LCD, bulging capacitors).
